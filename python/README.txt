====== vkontakte-getter ======

Программа загружает странички друзей с сайта http://vkontakte.ru в каталог data и создает страницу result.html с картой друзей, статистикой, интересами и группами. 
При желании пользователя программа автоматически выложит полученную страничку на сайт http://vkontakte.net.ru/номер , где номер - это уникальный номер пользователя с сайта vkontakte.ru.

===== Установка и использование =====

Если вы скачали архив с программой, распакуйте его и запустите vkontakte_getter.exe.
Введите свой email и пароль, которые вы используете для входа на сайт http://vkontakte.ru. 
Снимите флаг "загрузить страницу на сайт vkontakte.net.ru", если вы не хотите публиковать полученную страницу.
Программа создаст страницу result.html и откроет ее в браузере.
Если в Internet Explorer карта не видна, включите ActiveX.

Если вы скачали программу в виде архива с исходниками, то для запуска вам понадобится Python, его можно скачать с сайта http://www.python.org 
Если вы используете ОС Linux, то скорее всего Python у вас есть, однако вам может потребоваться установить пакет python-tk. В Ubuntu и Debian это делается так: sudo apt-get install python-tk

Для запуска напишите в командной строке:
python vkontakte_getter.py
